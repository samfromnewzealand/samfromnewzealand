This directory contains favicon files in various sizes and formats.

To generate a complete set of favicons:
1. Start with a square image (at least 512x512px)
2. Use a tool like https://realfavicongenerator.net/ to create all required sizes
3. Place all generated files in this directory
4. The base.njk template already includes references to these files

Common favicon files include:
- favicon.ico (16x16, 32x32)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- android-chrome-192x192.png
- android-chrome-512x512.png
- site.webmanifest
